This Plugin gets the weather from google.de/google.com and shows the weather now and a 4 days forecast with tiny pics

== Description ==

This Plugin is very easy to handle. Just install with a few clicks and set it up in your configs. 
Then include it in your sidebar e.g. with the following code:

< ?php spWeatherShow(); ? > (without spaces)

== Installation ==

1. Upload spWeather to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust the options
